# 打字练习

一个使用 Flutter 构建的学习和练习打字的网站。

## 特性

- 实时速度 (每分钟字数) 计算。
- 准确率跟踪。
- 正确和错误按键的视觉反馈。
- 一个简洁、专注的用户界面。
